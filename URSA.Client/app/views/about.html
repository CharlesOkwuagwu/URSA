This is a demo of Ultimate ResT API (URSA - the bear). URSA is composed of two components:

  • .net server side ReST API alternative with self documentation abilities base on HYpermedia DRiven Application (Hydra) Core Vocabulary.
  • angularJS client side ReST API that utilises an API documentation generated by the server side, builds necessary views on the fly and runs the machiner in motion.
